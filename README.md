# Nuxt Minimal Starter

> [!NOTE]
> Se recomienda instalar la extensión **Vue - Official** en VS Code para obtener soporte completo de TypeScript, autocompletado y resaltado de sintaxis en los archivos `.vue`.

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.


## Setup

Make sure to install dependencies:

```bash
# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm dev
```
