# Proyecto Web Frontend

Proyecto desarrollado con Nuxt.

## Requisitos Previos e Instalación

### 1. Clonar el repositorio
Para obtener una copia local del proyecto, clona este repositorio en tu máquina:

```bash
git clone https://github.com/PaoloESAN/Proyecto-Web-Frontend.git
cd Proyecto-Web-Frontend
```

### 2. Instalar pnpm
En este proyecto utilizamos **pnpm** como gestor de paquetes. Lo preferimos y recomendamos porque es **muchísimo más seguro** que npm.

Si aún no tienes instalado `pnpm`, puedes instalarlo de manera global ejecutando:

```bash
npm install -g pnpm
```
Si no tienes npm, para instalarlo, ve a google e instala **Nodejs**

### 3. Instalar las dependencias
Una vez dentro del directorio del proyecto, instala las dependencias con:

```bash
pnpm install
```

## Servidor de Desarrollo

### 4. Iniciar el proyecto
Inicia el servidor de desarrollo local ejecutando:

```bash
pnpm dev
```

### 5. Probar en el navegador
Una vez que el servidor de desarrollo comience a correr, abre tu navegador preferido e ingresa a:

👉 **[http://localhost:3000](http://localhost:3000)**

---

> [!NOTE]
> Se recomienda instalar la extensión **Vue - Official** en VS Code para obtener soporte completo de TypeScript, autocompletado y resaltado de sintaxis en los archivos `.vue`.
